How to Run the Project Locally

Download the project and run the Streamlit app with the following command (replace the file path with your own):

streamlit run "c:\Users\avach\Downloads\tech saksham disease prediction\disease_prediction.py"


After replacing the path according to your system, the project will open automatically in your browser on localhost.
